# 分镜脚本创作指南

本文档提供专业分镜脚本的创作方法和模板。

---

## 一、分镜规划要素

### 1.1 叙事结构分析

根据图片数量和用户意图，选择合适的叙事结构：

| 结构类型     | 适用图片数 | 特点               | 适用场景             |
| ------------ | ---------- | ------------------ | -------------------- |
| **三幕式**   | 3-5 张     | 起承转合           | 短故事、情节片段     |
| **蒙太奇式** | 5-9 张     | 视觉关联、主题统一 | MV、广告、情绪短片   |
| **故事式**   | 7-12 张    | 完整叙事线         | 短片、故事视频       |
| **展示式**   | 3-6 张     | 无叙事、纯展示     | 产品、风景、人物介绍 |

### 1.2 景别分布规划

#### 经典景别节奏公式

```
远景 → 中景 → 近景 → 特写 → 中景 → 远景
(建立)  (交代)  (聚焦)  (高潮)  (缓冲)  (结束)
```

#### 景别功能

| 景别      | 叙事功能           | 通常位置         |
| --------- | ------------------ | ---------------- |
| 远景/全景 | 建立场景、交代环境 | 开头、转场后     |
| 中景      | 叙事推进、人物互动 | 中间、对话       |
| 近景      | 情感聚焦、细节强调 | 情感点、重要时刻 |
| 特写      | 高潮强调、关键信息 | 关键点、转折     |

### 1.3 时长分配建议

| 镜头类型      | 建议时长 | 说明           |
| ------------- | -------- | -------------- |
| 远景/建立镜头 | 3-5 秒   | 让观众理解环境 |
| 中景          | 2-4 秒   | 标准叙事节奏   |
| 近景/特写     | 1-3 秒   | 短促有力       |
| 动作镜头      | 2-4 秒   | 根据动作复杂度 |
| 情感镜头      | 3-5 秒   | 留给观众消化   |
| 过渡镜头      | 1-2 秒   | 快速连接       |

## 二、叙事结构模板

### 2.1 三幕式结构 (3-5 张图)

```
第一幕: 开端 (1-2 镜头)
├── 镜头 1: 远景 - 建立场景
└── 镜头 2: 中景 - 引入人物

第二幕: 发展 (1-2 镜头)
├── 镜头 3: 近景 - 冲突/情感展开
└── 镜头 4: 特写 - 高潮点

第三幕: 结局 (1 镜头)
└── 镜头 5: 远/中景 - 解决或开放式结尾
```

### 2.2 蒙太奇式结构 (5-9 张图)

```
无明确叙事线，通过以下方式统一：
- 主题统一 (同一情感/概念)
- 视觉统一 (色调/构图相似)
- 节奏统一 (时长/转场一致)
- 音乐统一 (配乐驱动节奏)

示例结构:
镜头1 → 镜头2 → 镜头3 → ... → 镜头N
  ↓        ↓        ↓           ↓
[主题A] [主题A'] [主题A''] ... [主题A回归]
```

### 2.3 故事式结构 (7-12 张图)

```
1. 开场建立 (远景)     - 环境、时间
2. 人物介绍 (中景)     - 主角登场
3. 日常状态 (中景)     - 正常生活
4. 事件触发 (近景)     - 打破平衡
5. 发展 1 (中景)       - 应对变化
6. 发展 2 (近景)       - 深入冲突
7. 高潮 (特写)         - 情感顶点
8. 转折 (近景)         - 变化发生
9. 解决 (中景)         - 问题解决
10. 新平衡 (远景)      - 新的状态
11. 尾声 (中/远景)     - 余韵
```

## 四、分镜工作流程

### Step B1: 分镜规划

```
输入: 多张图片 + 用户描述
处理:
1. 统计图片数量
2. 分析图片内容，推断叙事顺序
3. 识别情感基调
4. 选择叙事结构类型
5. 分配每镜头景别
6. 规划时长分布
7. 选择转场类型

输出: 分镜结构表
```

### Step B2: 循环生成

```
对每张图片执行:
1. 分析画面内容和约束
2. 根据分镜规划的景别要求选择运镜
3. 生成符合规范的提示词

额外添加:
- 镜头编号
- 转场标注
- 时长标注
- Image 编号（多人场景）
```

### Step B3: 整合输出

```
按分镜模板格式化:
1. 标题和元数据
2. 按顺序排列所有镜头
3. 添加整体备注
4. 验证检查（Image 编号、语言保留等）
```

## 五、场景-镜头匹配参考

### 5.1 按场景类型

| 场景类型     | 推荐开场 | 推荐发展   | 推荐高潮 | 推荐结尾 |
| ------------ | -------- | ---------- | -------- | -------- |
| **爱情故事** | 远景环境 | 双人中景   | 特写对视 | 双人远景 |
| **动作追逐** | 俯拍全景 | 跟拍中景   | POV/FPV  | 远景结果 |
| **风景展示** | 航拍远景 | Pan 横移   | 特写细节 | 延时远景 |
| **产品介绍** | 全景环境 | Orbit 环绕 | 特写细节 | 使用场景 |
| **人物访谈** | 中景介绍 | OTS 对话   | 特写情感 | 远景收尾 |

### 5.2 按情感类型

| 情感类型     | 整体节奏       | 转场偏好   | 运镜偏好     |
| ------------ | -------------- | ---------- | ------------ |
| **欢快活泼** | 快节奏、短镜头 | 硬切为主   | 动态、跟拍   |
| **悲伤沉重** | 慢节奏、长镜头 | 叠化、淡出 | 缓慢推拉     |
| **紧张悬疑** | 不规则、突变   | 硬切、突切 | 推进、荷兰角 |
| **浪漫温馨** | 中等、流畅     | 叠化       | 柔和弧形     |
| **震撼史诗** | 宏大、层层推进 | 硬切+叠化  | 航拍、升降   |

## 六、分镜示例

### 示例：3 镜头告别场景

```markdown
# 告别 分镜脚本

**总时长:** 10 秒 | **镜头数:** 3 | **风格:** 文艺/慢节奏
**目标平台:** Runway Gen-4

---

## 镜头 1 | 4秒 | 远景
**转场:** 淡入 (Fade In)
**运镜:** Slow Crane Down

**提示词:**
```
Melancholic establishing shot, train station at dusk.

Camera Movement: Crane slowly descends through mist, revealing the empty platform, steady cinematic movement.

Subject & Action: Image 1 stands alone at the edge of platform, silhouette against golden light, motionless.

Environment & Mood: Golden hour sunlight streaming through windows. Steam rising from tracks. Nostalgic, bittersweet atmosphere.
```

---

## 镜头 2 | 3秒 | 近景
**转场:** 叠化 (Dissolve)
**运镜:** Gentle Dolly In

**提示词:**
```
Emotional close-up, intimate farewell moment.

Camera Movement: Camera gently pushes toward Image 1's face, subtle handheld movement adding intimacy.

Subject & Action: Image 1 turns slightly, eyes glistening, whispers "再见" with a small, sad smile.

Environment & Mood: Soft backlight creates rim lighting around hair. Shallow depth of field. Warm but melancholic.
```

---

## 镜头 3 | 3秒 | 远景
**转场:** 叠化 → 淡出
**运镜:** Static → Slow Zoom Out

**提示词:**
```
Poignant ending shot, loneliness emphasized.

Camera Movement: Static shot slowly zooms out, isolating the figure on the vast platform.

Subject & Action: Image 1 watches as unseen train departs, hand slowly lowering from a wave.

Environment & Mood: Train platform empties. Golden light fades to cooler tones. Fade to black. Sense of ending and emptiness.
```

---

## 整体备注

- **音乐建议:** 钢琴独奏，慢节奏，小调
- **色调统一:** 暖金色主调，渐变为冷青色
- **情感弧线:** 平静 → 情感释放 → 孤独
```
