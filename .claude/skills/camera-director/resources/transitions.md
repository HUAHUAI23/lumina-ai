# 电影转场类型指南

本文档包含专业电影转场类型，用于分镜脚本中的镜头衔接设计。

---

## 一、基础转场 (Basic Transitions)

### 1.1 切 (Cut)

最基本、最常用的转场方式，一个镜头直接切换到下一个镜头。

| 类型       | 英文            | 效果描述           | 叙事作用           | 使用场景                 |
| ---------- | --------------- | ------------------ | ------------------ | ------------------------ |
| **硬切**   | Hard Cut        | 直接切换，无过渡   | 推进情节、维持节奏 | 最常用，对话、动作、叙事 |
| **跳切**   | Jump Cut        | 同场景内跳跃       | 时间压缩、紧迫感   | 快节奏剪辑、时间流逝     |
| **L 切**   | L-Cut           | 画面先切，声音延续 | 平滑过渡、连接情感 | 对话场景、情绪延续       |
| **J 切**   | J-Cut           | 声音先切，画面后切 | 预示下一场景、铺垫 | 悬念、期待、转场预告     |
| **动作切** | Match on Action | 动作中切换         | 流畅、连贯         | 打斗、运动、动态场景     |

### 硬切 (Hard Cut)
```
使用时机: 默认选择，适合大多数场景
效果: 干净利落，不引起注意
标注: [Cut] 或省略不标注
```

### 跳切 (Jump Cut)
```
使用时机: 同一场景内的时间跳跃
效果: 打破连续性，产生不安或紧迫感
标注: [Jump Cut]
示例: 人物准备出门 → [Jump Cut] → 人物已在路上
```

## 二、渐变转场 (Dissolve Transitions)

### 2.1 叠化 (Dissolve)

两个画面重叠渐变，常用于表现时间流逝或情感连接。

| 类型         | 英文           | 效果描述       | 叙事作用       | 使用场景           |
| ------------ | -------------- | -------------- | -------------- | ------------------ |
| **标准叠化** | Dissolve       | 两画面渐变重叠 | 时间流逝、回忆 | 时空转换、蒙太奇   |
| **长叠化**   | Long Dissolve  | 延长的重叠时间 | 深层连接、梦幻 | 诗意转换、情感高潮 |
| **交叉叠化** | Cross Dissolve | 对等重叠       | 并列叙事       | 平行场景、对比     |

```
标注: [Dissolve] 或 [Cross Dissolve]
时长建议: 0.5-2 秒
适用: 时间跳跃、回忆片段、场景情感连接
```

### 2.2 淡入淡出 (Fade)

画面渐变至纯色（通常是黑色或白色）或从纯色渐现。

| 类型         | 英文          | 效果描述        | 叙事作用             | 使用场景           |
| ------------ | ------------- | --------------- | -------------------- | ------------------ |
| **淡入**     | Fade In       | 从黑/白渐现画面 | 开始、觉醒、新章节   | 开场、段落开始     |
| **淡出**     | Fade Out      | 画面渐变至黑/白 | 结束、沉睡、章节结束 | 结尾、段落结束     |
| **淡出淡入** | Fade to Black | 淡出后再淡入    | 时间大跨度、分隔     | 章节分隔、时间跳跃 |
| **白闪**     | Fade to White | 淡出至白色      | 死亡、回忆、升华     | 梦境、记忆、高光   |

```
标注: [Fade In] / [Fade Out] / [Fade to Black]
时长建议: 1-3 秒
适用: 开头结尾、章节分隔、情绪高潮
```

## 三、特殊转场 (Special Transitions)

### 3.1 擦除 (Wipe)

新画面像"擦"过一样替代旧画面。

| 类型         | 英文            | 效果描述   | 叙事作用   | 使用场景         |
| ------------ | --------------- | ---------- | ---------- | ---------------- |
| **横向擦除** | Horizontal Wipe | 左右擦过   | 场景切换   | 星球大战风格     |
| **纵向擦除** | Vertical Wipe   | 上下擦过   | 时间转换   | 复古风格         |
| **对角擦除** | Diagonal Wipe   | 斜角擦过   | 动态转换   | 活泼、运动       |
| **圆形擦除** | Iris Wipe       | 圆圈缩放   | 复古、喜剧 | 老电影风格、卡通 |
| **星形擦除** | Star Wipe       | 星形展开   | 夸张、喜剧 | 综艺、儿童节目   |
| **时钟擦除** | Clock Wipe      | 时钟式旋转 | 时间概念   | 时间相关转场     |

```
标注: [Wipe] / [Iris Wipe]
风格: 较为复古或风格化，现代影片慎用
经典: 《星球大战》系列大量使用
```

### 3.2 匹配剪辑 (Match Cut)

通过视觉或听觉相似性连接两个不同场景。

| 类型         | 英文          | 效果描述      | 叙事作用   | 使用场景 |
| ------------ | ------------- | ------------- | ---------- | -------- |
| **图形匹配** | Graphic Match | 形状/颜色相似 | 象征、隐喻 | 艺术表达 |
| **动作匹配** | Action Match  | 动作连贯      | 流畅过渡   | 动作场景 |
| **声音匹配** | Sound Match   | 声音跨场景    | 听觉连接   | 叙事桥接 |

```
经典案例:
- 《2001太空漫游》: 骨头 → 太空船 (Graphic Match)
- 动作片: 挥拳 → 落点 (Action Match)
标注: [Match Cut] + 描述匹配元素
```

## 四、音频转场 (Audio Transitions)

| 类型         | 英文             | 效果描述               | 使用场景         |
| ------------ | ---------------- | ---------------------- | ---------------- |
| **声音桥接** | Sound Bridge     | 下一场景的声音提前出现 | 预示转场         |
| **音乐过渡** | Music Transition | 音乐跨越场景           | 蒙太奇、时间流逝 |
| **声音匹配** | Sound Match      | 相似声音连接场景       | 创意转场         |
| **静默**     | Silence          | 突然安静               | 强调、反差       |

## 五、分镜脚本中的转场标注

### 标注格式

```markdown
## 镜头 N | 时长 | 景别
**转场:** [转场类型]
**运镜:** [运镜描述]
```

### 转场符号速查

| 符号              | 含义                 |
| ----------------- | -------------------- |
| `→ [Cut] →`       | 硬切                 |
| `→ [Dissolve] →`  | 叠化                 |
| `→ [Fade Out]`    | 淡出（通常用于结尾） |
| `[Fade In] →`     | 淡入（通常用于开头） |
| `→ [Wipe] →`      | 擦除                 |
| `→ [Match Cut] →` | 匹配剪辑             |

### 示例

```markdown
## 镜头 1 | 3秒 | 远景
**转场:** 淡入 (Fade In)
**运镜:** Slow Crane Down
...

→ [Dissolve] →

## 镜头 2 | 2秒 | 近景
**转场:** 叠化 (Dissolve)
**运镜:** Gentle Dolly In
...

→ [Cut] →

## 镜头 3 | 4秒 | 中景
**转场:** 硬切 (Cut)
**运镜:** Tracking Shot
...

→ [Fade Out]

## 结束
```

## 六、转场选择指南

### 按叙事需求选择

| 叙事需求      | 推荐转场                | 避免使用      |
| ------------- | ----------------------- | ------------- |
| **时间流逝**  | Dissolve, Fade to Black | Hard Cut      |
| **紧张节奏**  | Hard Cut, Jump Cut      | Long Dissolve |
| **梦境/回忆** | Dissolve, Fade to White | Hard Cut      |
| **平行叙事**  | Cross Dissolve, Wipe    | -             |
| **章节分隔**  | Fade to Black           | Dissolve      |
| **动作连贯**  | Match on Action         | Dissolve      |
| **开场建立**  | Fade In                 | Jump Cut      |
| **结尾收束**  | Fade Out, Dissolve      | Wipe          |

### 按风格选择

| 风格          | 常用转场                  |
| ------------- | ------------------------- |
| **现代电影**  | Hard Cut, Dissolve, Fade  |
| **复古/怀旧** | Wipe, Iris, Dissolve      |
| **纪录片**    | Hard Cut, L-Cut, J-Cut    |
| **MV/广告**   | 各种创意转场              |
| **恐怖/悬疑** | Fade to Black, Sudden Cut |
| **浪漫/文艺** | Dissolve, Slow Fade       |
